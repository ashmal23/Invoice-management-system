# 🚀 CryptoVault — Real-Time Cryptocurrency Dashboard

![CryptoVault Banner](https://i.ibb.co/8gh5Jc8/image.png)

A premium, modern cryptocurrency tracking application built with React.js featuring real-time data, interactive charts, portfolio tracking, and crypto news — all wrapped in a stunning dark neon UI.

## ✨ Features

- 📊 **Real-Time Crypto Stats** — Live prices, market cap, volume & global statistics
- 📈 **Interactive Price Charts** — Historical price data with multiple time period options
- 📰 **Crypto News Feed** — Latest cryptocurrency news from trusted sources
- 💱 **Exchange Information** — Detailed exchange data and rankings
- ⭐ **Watchlist** — Save your favorite cryptocurrencies for quick access
- 💰 **Portfolio Tracker** — Track your crypto holdings and profit/loss in real-time
- 🔥 **Trending Coins** — Animated ticker showing top performing coins
- 🔍 **Search & Filter** — Quickly find any cryptocurrency
- 📱 **Fully Responsive** — Works perfectly on desktop, tablet & mobile
- 🌙 **Premium Dark Theme** — Stunning neon glassmorphism UI

## 🛠️ Tech Stack

- **Frontend:** React.js, Ant Design, Chart.js
- **State Management:** Redux Toolkit (RTK Query)
- **APIs:** CoinRanking API, Bing News API (via RapidAPI)
- **Styling:** Custom CSS with Glassmorphism & Neon Gradients

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/ashmal23/Invoice-management-system.git

# Install dependencies
npm install

# Start the development server
npm start
```

## 📁 Project Structure

```
src/
├── app/
│   └── store.js          # Redux store configuration
├── components/

│   ├── CryptoDetails.jsx    # Individual coin details
│   ├── News.jsx           # Crypto news feed
│   ├── Exchanges.jsx      # Exchange information

│   ├── TrendingCoins.jsx  # Trending ticker

├── services/
│   ├── cryptoApi.js       # CoinRanking API integration
│   └── cryptoNewsApi.js   # News API integration
└── App.js                 # Main app with routing
```
